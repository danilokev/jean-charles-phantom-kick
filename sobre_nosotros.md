# Nombre del juego: Jean Charles Phantom Kick

Las trepidantes aventuras de Jean Charles en su nuevo título: Phantom Kick. Un Shoot 'Em Up de ritmo frenético donde esquivar balas se vuelve un infierno. Con controles sencillos pero desafiantes de dominar, te enfrentarás a hordas de enemigos que te lanzarán cortinas de proyectiles espectaculares. Vive una historia dinámica contada a través de diálogos vibrantes entre personajes, mientras mejoras tus reflejos en cada partida. Domina los patrones de ataque en el Modo Historia o perfecciona tus habilidades en el Modo Práctica. Con gráficos llamativos, una sensación de velocidad imparable y un sistema de puntuación que desafía a los más competitivos, este juego te mantendrá pegado a la pantalla. ¿Listo para el reto?

# Comentarios sobre la práctica que se consideren oportunos

Para trabajar en git con la rama experimental, hacer ramas en local, no trabajar directamente en experimental:

- Crear una rama distinta:

```sh
git checkout -b {rama}
```

- Al terminar el prototipo, volver a la rama experimental

```sh
git checkout experimental
```

- Asegurarse de que la rama está actualizada

```sh
git pull
```

- Unir la rama prototipo

```sh
git merge {rama}
```

- Subir a GitHub inmediatamente para evitar dos ramas experimental locales distintas

```sh
git push origin experimental
```
